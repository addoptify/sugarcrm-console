sugarcrm-repair
===============

Makes it possible to execute the Sugar CRM quick repair and rebuild actions trough the command line

Installation
---------------------

```sh
git clone https://github.com/kilhage/sugarcrm-repair sugarcrm-repair
cd sugarcrm-repair
sudo rake install
```

Usage
---------------------

```sh
cd /path/to/sugarcrm
sugarcrm-repair
```
